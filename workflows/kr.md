# 한국어 명령어 실행 워크플로우 (/kr)

사용자가 한국어로 명령을 내리거나 `@키워드` 형태로 요청할 때 이 워크플로우를 따라 실행합니다.

## 실행 절차

1. **명령어 확인**: 사용자 입력에서 동일 폴더 내의 `../korean_commands.json`에 정의된 키워드를 찾습니다.
2. **매핑 타입 확인**:
   - `type: "skill"`인 경우: 해당 스킬 ID를 사용하여 `view_file`로 `SKILL.md`를 읽고 지침을 따릅니다.
   - `type: "mcp"`인 경우: 해당 MCP 서버와 도구를 확인하고 사용자에게 실행을 제안하거나 직접 실행합니다.
3. **지능적 매핑**: 키워드가 정확히 일치하지 않더라도 의도가 분명한 경우(예: "그림 그려줘" -> "이미지생성") 가장 유사한 명령어를 추천합니다.
4. **결과 보고**: 실행 결과를 한국어로 친절하게 보고합니다.

## 주요 사용 예시

- **@디자인**: `frontend-design` 스킬을 활성화하여 UI 디자인 작업을 시작합니다.
- **@검색**: 최신 정보가 필요한 경우 `search_web` 도구를 사용하여 검색을 수행합니다.
- **@이미지생성**: 사용자의 묘사를 바탕으로 `stitch.generate_screen_from_text`를 실행합니다.
